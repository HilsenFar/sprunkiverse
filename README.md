# Jason's Sprunkiverse

A fan-made 3D Sprunki universe in the browser - music mixer, open world, god simulator and aquarium.

Play: https://spil.gitato.net

Original Sprunki characters & sounds by NyankoBfLol (Scratch project, CC BY-SA 2.0).
Built with three.js and the Web Audio API. Fan project, non-commercial.
